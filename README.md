# LearnEdge Platform 🐧🚀

An AI-powered coding learning platform that helps students write, run, and improve code using real-time feedback.

---

## 🔥 Features

- 💻 Code Editor + Compiler (Judge0)
- 🧠 AI Code Review (LLM - Groq)
- 🐧 Interactive Mascot (Pingo)
- 🌐 Multi-language support (JS, Python, C++, Java)

---

## 🏗️ Project Structure

frontend/ → React app (UI, editor, mascot)  
llm_service/ → Python FastAPI (AI code review engine)
